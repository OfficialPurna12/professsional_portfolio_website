# Purna Bahadur Khatri - Portfolio Website

A modern, responsive, and professional portfolio website built with HTML5, CSS3, JavaScript, and Bootstrap 5. Features dark mode toggle, WhatsApp integration, certificate modals, and dynamic image loading.

## ✨ Features

### Core Features
- **Responsive Design** - Mobile-first approach using Bootstrap 5
- **Dark/Light Mode Toggle** - Persistent theme preference with localStorage
- **WhatsApp Integration** - Floating chat button and contact option
- **Certificate Viewer** - Modal popup for viewing achievement certificates
- **Smooth Animations** - Scroll-triggered fade-ins and hover effects
- **Form Validation** - Client-side validation with visual feedback

### Sections
1. **Hero Section** - Professional introduction with animated photo frame
2. **About Section** - Bio, education timeline with hover effects
3. **Skills Section** - Animated progress bars and skill cards
4. **Portfolio Section** - 6 project cards with image support and overlay effects
5. **Achievements Section** - Certificates with view buttons
6. **Contact Section** - Validated form and social links

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and SEO optimization
- **CSS3** - Custom properties, animations, and responsive design
- **JavaScript (ES6+)** - Interactivity, dark mode, and form validation
- **Bootstrap 5.3.2** - Grid system and components
- **Bootstrap Icons** - Modern icon library
- **Google Fonts** - Poppins and Inter typography
